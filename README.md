# Нейрофизиология пранаямы — Neuro-Universe

Лендинг авторского курса **Леонида Кутузова** «Нейрофизиология пранаямы» — третьего курса экосистемы [neuro-universe.ru](https://neuro-universe.ru), прямого продолжения программ «Нейрофизиология» и «Нейрофизиология медитации».

## Стек

Статический HTML/CSS, без сборки и зависимостей. Шрифты — Cormorant Garamond (display) + Inter (body) + JetBrains Mono (служебные подписи) с Google Fonts.

## Структура

| Файл | Назначение |

## Структура лендинга

1. **Hero** — главный экран с фотопортретом, датами и CTA.
2. **Контекст экосистемы** — линия из трёх курсов проекта, объяснение «соматического моста».
3. **Манифест** — дыхание как язык нервной системы (с фото-акцентом).
4. **Аудитории** — три карточки: преподаватели, психологи, предприниматели.
5. **Программа** — 3 модуля × 4 темы (12 тем) с академическими аннотациями.
6. **Практика** — 12 практических классов с нейрофизиологическими описаниями.
7. **Научные основания** — 6 рецензируемых публикаций (Frontiers, PNAS, ERS Breathe, ScienceDirect) + SVG-визуализация ритмов ЭЭГ.
8. **Результаты** — 6 тезисов о выходе с курса.
9. **Об авторе** — портреты Леонида Кутузова и био.
10. **Формат и цифровая гигиена** — GetCourse, без чатов в Telegram.
11. **Тарифы** — Базовый / Профессиональный / Всё включено.
12. **FAQ** — 3 развёрнутых ответа.

## Локальная разработка

Просто откройте `index.html` в браузере — никаких сборщиков и серверов не требуется. Для удобства разработки можно поднять любой статический сервер:

```bash
# Python
python3 -m http.server 8000

# или Node.js (npx)
npx serve .
```

Затем — http://localhost:8000.

## Tone of Voice

Файл [`.cursorrules`](.cursorrules) фиксирует правила копирайта: академический, сдержанный, премиальный регистр без капслока, восклицательных знаков и инфобизнесовых клише; уважительная подача традиции йоги и буддизма; корректная нейрофизиологическая терминология. Cursor автоматически учитывает эти правила при правке файлов в пределах папки.
